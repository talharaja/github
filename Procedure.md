## Steps for setting Github:  
Use below commands after creating a folder on local machine.  
	`git init `  
	`git branch -m main `  
	`git add .`  
	`git commit -m "first commit"`  

Then, create repo on Github.  
	`git remote add origin https://url.git    `
	`git push -u origin main`  

Additional commands:
	`git status`
	`git log`
	`git remote -v`
